# 🌍 HumRahi Adventures

An AI-powered travel booking platform that intelligently matches users with the best vendors based on budget, location, and preferences.

---

## 🚀 Features

* 🤖 AI-based trip recommendation
* 🏨 Smart vendor matching system
* 📅 Real-time availability check
* 💳 Online payment integration
* 📩 Email & WhatsApp notifications
* ⭐ Review & rating system
* 🛠️ Admin dashboard
* 👨‍💼 Vendor dashboard

---

## 🧠 Tech Stack

### Frontend

* Next.js
* React
* Tailwind CSS

### Backend

* Node.js
* Prisma ORM
* PostgreSQL

### Integrations

* Razorpay (Payments)
* Nodemailer (Email)
* Twilio (WhatsApp/SMS)
* OpenAI API (AI Recommendations)

```bash

cd HumRahi-Adventures
npm install
npm run dev
```
